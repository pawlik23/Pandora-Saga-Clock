# MMO Clock (Expo)

A simple MMO in-game clock app with day/night countdown.

- 1h in-game = 7m30s real life (8x speed)
- Night = 11:00 to 21:00 in-game
- Shows current game time and countdowns

## Run locally
```bash
npm install
npx expo start
```

## Build APK
```bash
npx expo login
npx expo build:android -t apk
```
